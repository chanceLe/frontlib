# 常用函数

## 随机码

### 随机字符串

```js
function getUniqueStr(n = 13) {
  return Math.random()
    .toString(16)
    .substring(2, n + 2);
}
```

## 工程化相关

### 引入 CDN

```js
export const importCDN = (url, name) =>
  new Promise(resolve => {
    const dom = document.createElement('script');
    dom.src = url;
    dom.type = 'text/javascript';
    dom.onload = () => {
      resolve(window[name]);
    };
    document.head.appendChild(dom);
  });
```
